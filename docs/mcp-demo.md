# MCP統合のデモ

このファイルはModel Context Protocol (MCP)を使用してGitHub APIを呼び出し、作成されました。

## MCPとは

Model Context Protocol (MCP)は、大規模言語モデル(LLM)と外部サービスを連携させるためのプロトコルです。
この例では、MCPを使用してGitHubのAPIを呼び出し、以下の操作を行っています：

1. 新しいブランチの作成
2. このマークダウンファイルの追加
3. プルリクエスト（マージリクエスト）の作成

## 宣言的アプローチとの関連性

このリポジトリは宣言的UIについての資料を含んでいますが、MCPの使用も宣言的なアプローチの一例と言えます。
MCPを介してGitHubのAPIを呼び出す際は、「どのような結果が欲しいか」を宣言的に指定し、
その実装詳細はMCPプロトコルとGitHub APIが処理してくれます。
