# Column 2: マークダウンの書き方Tips

> 前回のコラムでマークダウンの基本を紹介しました。今回は実践的なTipsを紹介します。

## よく使う記法チートシート

### 見出し

```markdown
# H1 大見出し
## H2 中見出し
### H3 小見出し
```

### 強調

```markdown
**太字**
*斜体*
~~取り消し線~~
`インラインコード`
```

### リスト

```markdown
- 箇条書き1
- 箇条書き2
  - ネストもできる

1. 番号付き
2. リスト
```

### リンクと画像

```markdown
[リンクテキスト](https://example.com)
![代替テキスト](image.png)
```

### コードブロック

````markdown
```javascript
const hello = "world";
```
````

### テーブル

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```

## Tips

### Tip 1: 空行を忘れずに

段落を分けるには**空行**が必要です。

```markdown
これは段落1。

これは段落2。
```

### Tip 2: リストの前後にも空行

```markdown
本文テキスト。

- リスト1
- リスト2

続きのテキスト。
```

### Tip 3: コードブロックには言語を指定

シンタックスハイライトが効きます。
