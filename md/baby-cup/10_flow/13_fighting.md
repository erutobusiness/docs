# 対戦

## Appendix: 実況・解説のヒント

このSDLCの観点を持つと、実況や解説の深みが増します。状況に応じたトークネタの引き出しとして活用してください。

| 状況 | トークテーマ案 |
| :--- | :--- |
| **選出が完璧にハマった** | 「リソース配分が完璧ですね。オートスケーリングが効いています」 |
| **急所で負けた/事故った** | 「これは予期せぬ障害ですね。SRE力が試されます」 |
| **マイナーが活躍した** | 「制約があるからこそ生まれる創造性ですね。枯れた技術の水平思考です」 |

### 注意事項
- [ ] **バッファ時間の確保**: 対戦（デプロイ）は予期せぬトラブルで長引くものです。
- [ ] **遅延時の対応**: 話が盛り上がっても、タイムボックス（スケジュール）は意識しましょう。
