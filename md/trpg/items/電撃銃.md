# 電撃銃（ミゴの電気銃）

## 基本情報

| 項目 | 値 |
|------|-----|
| 名前 | 電撃銃（電気銃） |
| タイプ | 遠距離武器 |
| 起源 | ミゴの技術 |
| 入手難度 | 非常に困難（ミゴからのみ入手可能） |

## 外見・説明

黒い金属のかたまりのような外観の銃。
地球の技術とは明らかに異なる異質な構造をしている。
電撃を発射して対象を攻撃する。

## 効果・性能

### 戦闘データ

| 項目 | 値 |
|------|-----|
| ダメージ | 1D10（電撃） |
| 射程 | 10m |
| 攻撃回数 | 1 |
| 貫通 | なし |
| 基本命中率 | 25%〜55%（資料により異なる） |

### 特殊効果

| 効果 | 説明 |
|------|------|
| スタン | ダメージ分のラウンド、対象は行動不能になる |
| 心臓麻痺判定 | ダメージと耐久力（CON）のロールを行い、失敗すると心臓麻痺で即死 |

## 入手方法

ミゴから提供される、または倒したミゴから奪取。

## 使用条件

特別な訓練は不要だが、ミゴの技術のため人間には扱いにくい可能性あり。
