# 第1章：1日目

## タイムスケジュール

| 時刻   | イベント             |
| ------ | -------------------- |
| 朝8時  | 集合場所に集合、出発 |
| 昼12時 | 村に到着             |
| 15時頃 | 吊り橋を渡って村へ   |
| 18時   | 夕食、人狼伝説       |
| 夜     | 自由行動→ガイド殺害  |

## 村への到着

ツアーガイドの内藤に従い、挨拶などのレクリエーションや昼食を済ませると、15時頃に吊り橋の前に到着する。

内藤によると、村への入り口は、森を抜けた先の吊り橋しかないとのことなので、バスから降りると荷物を持って歩いていくことになるだろう。

吊り橋を渡る中で、勇気のある探索者はこの吊り橋がなければ……。

### 村長の対応

村長が運転手に明日は来なくて良い旨を伝える。

## 自由行動（18時まで）

18時の夕食までは自由に行動できる。

※ただし、事件が起こる前なので、あまり情報は与えない。

## 夕食と人狼伝説

夕食は村中央の会館に皆で集まって行われる。
内藤が食事を出し、準備ができたところで村長の村岡が怪談『人狼伝説』を始める。

🎵BGM03を再生。

### 人狼伝説の内容

- 人間に化けた狼である人狼が、夜な夜な人間を襲う
- 人狼は人間に完璧に化け、普通には見破ることはできない
- 人間は人狼に対抗するため、特殊な能力を身につけるものがいた
  - **看破の秘術**：人狼かどうかを見抜く
  - **降霊術**：死者の魂と会話できる
  - **庇護の術**：人狼の脅威を防ぐ
  - **裏切り者**：人狼の魔力に心奪われ手助けをする
- 村人たちは1日1回の投票によって怪しい人物を多数決で処刑することによって村を守ってきた
- かの有名な「人狼ゲーム」もこのような伝説が元になって生まれたとされている

### 村長のセリフ

> 改めまして、大神村へお越しいただき、心より歓迎いたしますのじゃ。
> 私がこの村の村長の村岡と申しますのじゃ。
> 人狼伝説という、興味深い話がございますので、どうぞお聞きくださいませ。
>
> この地には、かつて人狼と呼ばれる恐ろしい獣が存在しましたのじゃ。
> 人に化けることができ、夜ごとに人間を襲っていたと言われておりますのじゃ。
> 人狼は人間に完璧に化けるため、見分けるのはまず不可能なのですじゃ。
>
> しかし、人間はこの人狼に対抗するため、特殊な能力を身につけた者たちもおりました。
> 『看破の秘術』を使い人狼を見抜く者、『降霊術』を使って死者の魂と会話できる者、『庇護の術』で人狼の脅威から村人を守る者、そして、残念ながら人狼の魔力に取り込まれてしまい手助けをする『裏切り者』がいたのですじゃ。
>
> このような特殊な能力を駆使しながら、日ごとに投票を行い、怪しい人物を多数決で処刑することで村を守ってきました。
> この伝説が元になって、皆さんもご存じの「人狼ゲーム」が誕生したとも言われておるんですじゃ。
>
> 大神村の人狼伝説、皆様に心に残るものであれば幸いです。
> お帰りの際には、ぜひ村の観光スポットを訪れたり、お土産をお求めいただけると嬉しいです。
> どうぞ、大神村の魅力をお楽しみください。

村長の話を種にツアー参加者たちの会話も弾み、夜を迎える。

## 1日目夜

まだ事件が起きているわけでもないので、特別な理由がなければ探索はなし。

何かするとしても電気の通っていない廃村の夜。
判定は厳しめに。

### KP向け情報：ガイド殺害

時系列的には、このタイミングで村長グールが内藤の家を襲撃する。

1. 長刀で刺し殺した後、腹部に噛み傷を残す
2. グールは地下を移動するため、地上で張っていても遭遇不可
