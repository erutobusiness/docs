# 深き者の目的と手段

## 目的

**クトゥルフの復活**

## 手段

**星辰を正す**

## 方法：天使を利用する理由

### 天使の能力

十字教の天使は**天体制御（アストロインハンド）**と呼ばれる天体惑星を自在に操る力を持つとされている。

この力を利用すれば、星辰を正すことも可能なはず。

### 問題点

深き者の力では天使を直接操ることはできない。

### 解決策

1. 天使を弱体化させれば操れるかもしれない
2. 深き者はショゴスを使役できる
3. ショゴスは天使にはなれないが、**人を支配することはできる**
4. よって、**天使を人の身に堕とすことができれば**、天使を操ることも可能
