# SIREN

## 概要

探索者たちは抽選で当たったワイヤレスイヤフォン「SIREN」を使い始める。心地よい音源を聴き続けるうちに、ある夜、奇妙な夢の世界へと誘われる。そこはNPC「ロリ」の脳内を模した街であり、クトゥルフの企みを阻止するための戦いが始まる。

## タイトルの意味

「SIREN」には4つの意味が込められている：

1. **セイレーン** - ギリシャ神話の歌う怪物、9人のセイレーン
2. **サイレン** - 警報音、危険を知らせる音
3. **試練** - 探索者たちが乗り越えるべき試練
4. **紫恋** - （意味は未定）
