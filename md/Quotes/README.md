# 名言・格言コレクション

## 唯才是挙（ゆいざいぜきょ）

ただ才のみ是れ挙げよ。
ただ才あらば用いる。

出典: 曹操「唯才是挙令」

## 常套句を着こなす

> 人生の経験の大部分は常套句を覚えることである。
> 人間の教養の大部分は常套句を『着こなす』ことである。
> かくて模索。
> 就中高貴な模索であるべき青年の対話を、現代の青年は『合言葉』でしか話さない

出典: 三島由紀夫「詩論その他」（決定版全集26巻）

「AIの推論が実際には推論というよりパターンの記憶」だとする論文が話題になっているのを見て、青年に限らずAIまでもが常套句を着こなす時代なのかなど思ったりした。
