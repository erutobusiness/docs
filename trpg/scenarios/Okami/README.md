# 大神村の怪異

## 概要

探索者たちは「明け色の陽射し」主催の大神村ホラーツアーに参加する。
到着した翌朝、ツアーガイドの無残な死体が発見され、村長は「人狼伝説」に基づく処刑の儀を提案する。
しかし、真の敵は人狼ではなく、村に潜むグール（食屍鬼）たちであった。

## シナリオのキーポイント

- 人狼ゲームを隠れ蓑にしたグール（食屍鬼）による連続殺人
- NPCの大半はグールが人間に化けた姿
- 真の人間はNPC3名（内藤・大賀・尾上）のみ
- 村長の正体は9歳の魔術師幼女「小萌ナギサ」

## クリア条件

明確なクリア・終結点は設定されていない。

## 想定される脱出方法

- 1週間生存して郵便配達員に気づいてもらう
- 秘密によっては呪文や不死身の怪人の能力で脱出可能
- グールは知性的であり、交渉・説得・和解による解決も可能
