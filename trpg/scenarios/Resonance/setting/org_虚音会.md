# 虚音会（きょおんかい）

## 基本情報

|項目|値|
|-|-|
|名称|虚音会（きょおんかい）|
|拠点|横浜|
|設立|1920年頃|
|規模|約30名|
|表向きの顔|音響研究団体|
|管理する装置|低周波柱|

## 概要

虚音会は音響研究を装う秘密結社であり、6つのカルトの中で唯一「科学的アプローチ」を標榜している。
他のカルトが宗教的・神秘的に根源音を解釈するのに対し、虚音会は根源音を「制御可能な物理現象」と捉えている。

## 構成

### 幹部

- **会長**: 倉橋源蔵（くらはし げんぞう）、63歳。元帝国大学の音響学教授。戸叶と同じくウィーン留学経験を持つが、根源音を「人類の次の段階への進化の鍵」と信じている。学者的慎重さから計画を段階的に進めようとする
- **副会長**: 鳴瀬忠明（なるせ ただあき）、38歳。東京帝国大学出身の音響工学者。冷徹で合理的であり、根源音を純粋な科学研究の対象として捉えている。横浜拠点の実質的指揮官

### 構成員

- 音響工学の技術者: 約10名
- フロント企業（横浜蓄音機商会）の従業員: 約5名
- 港湾労働者（下級構成員）: 約15名

### 組織図

```
会長・倉橋源蔵
  ├─ 副会長・鳴瀬忠明
  │   ├─ 技術班（装置の研究・操作）
  │   └─ 情報班（他カルトとの通信）
  ├─ 横浜蓄音機商会（フロント）
  └─ 港湾班（地下施設の警備・労働）
```

## 活動

### 表の活動

- 蓄音機と音響部品の輸入販売
- 音響に関する勉強会の開催
- 横浜港の復興事業への参加

### 裏の活動

- 共鳴装置（低周波柱）の研究と段階的な起動
- 超低周波の人体実験（港湾労働者や浮浪者を使用）
- 他カルトとの国際通信と同時起動計画の策定
- 戸叶響一の殺害

## 虚音会と戸叶

虚音会は当初、戸叶の研究を利用しようとした。
戸叶が共鳴装置間の共鳴条件を理論的に解明したことが、同時起動計画の基礎となった。
しかし戸叶が計画に反対し妨害を試みたため、虚音会は彼を排除した。

### 殺害方法

横浜の共鳴装置を利用し、戸叶の書斎に向けて超低周波を集束させた。
装置と書斎の間に設置された「指向性増幅管」がこれを可能にした。
この増幅管は現在も横浜港地下に存在する。

### 指向性増幅管

共鳴装置の発する超低周波を特定方向に集束させる銅製の管である。
長さ2m、直径30cmで、出力側に古代文字が刻まれた音叉状の先端がある。
射程は最大500mに達する。
虚音会はこの装置を使って戸叶を遠隔から殺害した。

## 他カルトとの関係

虚音会は6つのカルトの中で最も近代的かつ組織的であり、同時起動計画の事実上の主導者である。
他のカルトとは貿易ルートや学術交流を装って通信している。

## 幹部の真の動機

### 倉橋源蔵：秩序への渇望

倉橋は1923年の関東大震災で妻と2人の子を失っている。
家屋の倒壊と火災の延焼、群衆の暴走が彼の人生を破壊した。
根源音に魅入られたのは、それが「科学で制御できる秩序」に見えたからである。
根源音の制御は、二度と無秩序に人生を奪われないための究極の保険であり、狂信というよりも絶望的な合理性に基づいている。

組織内では温厚な老紳士として振る舞うが、根源音の話題になると目に異様な光が宿る。

> 地震も津波も火災も、全ては制御の失敗だ。根源音を制すれば、我々は二度と無秩序に屈さずに済む

### 鳴瀬忠明：証明への執念

鳴瀬は学術界で正当な評価を得られなかった音響工学者である。
帝国大学での研究は「実用性がない」と却下され続けた。
虚音会の研究は自らの理論が正しかったことを証明する唯一の場であり、冷徹さの裏には「認められたい」という切実な欲求がある。

> 学会が私の論文を読む日が来る。その時には、根源音が全てを証明してくれるだろう

## 内部崩壊ルート

倉橋と鳴瀬の間には方針をめぐる不和がある。
鳴瀬は冷徹な合理主義者であり、倉橋の学者的慎重さを「非効率」と見なしている。
鳴瀬にとって倉橋は研究資金の源泉にすぎず、忠誠は信仰ではなく利害に基づいている。
探索者がこの対立を利用すれば、内部分裂を誘発できる。

具体的には以下の手段がありえる。

- 鳴瀬に「倉橋が計画を遅延させている」という偽情報を流す
- 倉橋に「鳴瀬が独断で装置を起動しようとしている」と密告する
- 双方の信頼する部下を買収または脅迫して、相互不信を増幅させる
