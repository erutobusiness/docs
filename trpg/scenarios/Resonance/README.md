# 残響世界 ── Resonance

## 概要

> 宇宙が生まれる前に「何か」が発した音 ── "根源音"（Ur-Klang）。
> その残響は今も次元の裂け目から漏れ続けている。
> 古代文明は独立してこの音を感知し、各地に「共鳴装置」を建造した。
> 1925年、6つの装置が同時に起動されようとしている。
> 全装置が共鳴すれば、根源音が現実に干渉し、次元の境界が崩壊する。
