# 脳缶（ブレイン・シリンダー）

## 基本情報

| 項目     | 値                                 |
| -------- | ---------------------------------- |
| 名前     | 脳缶（ブレイン・シリンダー）       |
| タイプ   | 保存・運搬装置                     |
| 起源     | ミゴの技術                         |
| 入手難度 | 非常に困難（ミゴからのみ入手可能） |

## 外見・説明

人間の脳を取り出して保存・運搬するための金属の筒。
ミゴの優れた外科技術の産物であり、内部の脳を生きたまま維持できる。

## 効果・性能

### 基本機能

- 人間の脳を取り出して安全に保存
- 長期間の保存・運搬が可能
- 脳は生きたまま意識を保つことができる

### 拡張機能（オプション装置）

| 装置     | 効果                                   |
| -------- | -------------------------------------- |
| 会話装置 | 脳缶の中の人間が外部と会話できる       |
| 視覚装置 | 脳缶の中の人間が外部を見ることができる |

## 入手方法

ミゴから提供される。
通常、ミゴは以下の場合に脳缶を使用する：

- 人間をユゴス（冥王星）へ運搬するため
- 実験・研究対象として人間の脳を保存するため
- 協力者との長期的な関係維持のため

## 使用条件

ミゴによる外科手術が必要。
脳の取り出しと脳缶への接続はミゴの専門技術。

- ミゴの宇宙船は人間の肉体を運ぶことができないため、地球外への移動には脳缶が必要
- 脳缶に入った人間は不老不死に近い状態となるとも言われる
