# 2. Claude Codeとは

## 概要

Claude CodeはAnthropicが提供する**CLI（コマンドラインインターフェース）ベース**のAIコーディングアシスタントです。

## 特徴

### ターミナルネイティブ

```bash
$ claude
```

ターミナルから直接起動し、会話形式でコーディングを進められます。

### エージェント的な動作

- ファイルの読み書き
- コマンドの実行
- プロジェクト全体の理解

を自律的に行います。

### コンテキストの深さ

長いコードベースでも、関連ファイルを探索して文脈を理解してくれます。

## 得意なこと

- 大規模なリファクタリング
- 複雑なバグの調査
- ドキュメントからの実装
- コードレビュー的なフィードバック
