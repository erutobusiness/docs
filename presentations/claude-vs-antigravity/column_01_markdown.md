# Column 1: マークダウンって何？

> このコラムでは、本編から少し離れて「マークダウン」についてお話しします。

## マークダウンとは

Markdown（マークダウン）は、**テキストを簡単に装飾するための記法**です。

2004年にJohn Gruberによって作られました。

## なぜマークダウンを使うのか

### 1. シンプル

特別なソフトがなくても、テキストエディタだけで書けます。

### 2. 読みやすい

装飾前の状態でも、人間が読んで意味がわかります。

```markdown
# これは見出し

これは **太字** で、これは _斜体_ です。

- リスト1
- リスト2
```

### 3. 変換しやすい

HTML、PDF、Word など、様々な形式に変換できます。

## AIとマークダウン

面白いことに、**AIはマークダウンが大好き**です。

- 構造化されているので理解しやすい
- コードブロックでプログラムを表現できる
- 見出しで文脈を区切れる

Claude CodeやGoogle Antigravityとやり取りするとき、マークダウンで指示を書くとうまくいくことが多いです。
