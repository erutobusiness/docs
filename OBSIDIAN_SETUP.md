# Obsidian Setup Guide

このドキュメントでは、Obsidian をより便利に使うための推奨設定とプラグインを紹介する。

## 推奨プラグイン (Community Plugins)

以下のプラグインを導入すると、ドキュメント管理が格段に楽になる。
導入については、「設定 > コミュニティプラグイン > 閲覧」から検索してインストールすること。

### 1. Dataview

- **用途**: データベース機能
- **メリット**: タグやフォルダを条件に、ファイル一覧を自動生成できる
- 例: `trpg` フォルダ内のすべてのNPCを一覧表にするなど
- **構文例**:
  ```dataview
  TABLE system, race
  FROM "trpg/characters"
  WHERE file.name != "_template"
  ```

### 2. Tag Wrangler

- **用途**: タグ管理
- **メリット**: タグパネルから右クリックで「タグ名の変更」が可能になる

### 3. Calendar

- **用途**: カレンダー表示
- **メリット**: サイドバーにカレンダーを表示し、日付ごとのメモ（デイリーノート）に素早くアクセスできる

### 4. Kanban

- **用途**: カンバンボード
- **メリット**: Trello のようなカード形式でタスク管理ができる

## テンプレートの使い方

1. 新しいノートを作成する (`Ctrl+N`)
2. コマンドパレットを開く (`Ctrl+P`)
3. `Insert template` と入力して選択する（または設定したホットキーを押す）
4. リストから使用したいテンプレート（例: `TRPG_PC_Template`）を選ぶ

## 画像の貼り付け

クリップボードの画像を `Ctrl+V` で貼り付けると、自動的に `assets/` フォルダに保存される。
ドキュメントフォルダが画像ファイルで埋め尽くされる心配はない。
