# 16Personalities と MBTI / Big Five

「16Personalities」は、MBTIのような4文字のアルファベットを用いるが、その裏側の理論は**ビッグファイブ（Big Five）**に基づいている独自の性格診断テストである。
正式名称は **NERIS Type Explorer** と呼ばれる。

## 1. 構造と違い

### 16Personalities (NERIS Type Explorer)

- **構造**: MBTIの指標（I/E, S/N, T/F, J/P）を採用しつつ、5つ目の指標として**Identity（アイデンティティ：-A / -T）**を追加している
- **理論**: ユングの心理学的タイプ論（心理機能）は**採用していない**
  代わりに、現代の心理学で主流の「ビッグファイブ（5因子モデル）」の各特性を、MBTIのアルファベットにマッピングして再構成している
- **特徴**: タイプ論（AかBか）の親しみやすさと、特性論（スペクトラム）の科学的根拠のハイブリッドを目指している

### MBTI (Myers-Briggs Type Indicator)

- **構造**: カール・ユングの理論に基づき、4つの二対立（指標）で16タイプに分類する
- **特徴**: 「心理機能（Cognitive Functions）」の優劣や順序を重視する
- **科学的評価**: 現代の学術的な心理学界では、信頼性（再テストで結果が変わる）や妥当性（現実に即しているか）の面で批判されることが多い

### ビッグファイブ (Big Five / OCEAN)

- **構造**: 開放性、誠実性、外向性、協調性、神経症的傾向の5つの因子で性格を記述する「特性論」
- **特徴**: 「タイプ」に分類するのではなく、「程度（スペクトラム）」で測定する
- **科学的評価**: 信頼性・妥当性が高く、現代の心理学研究における**ゴールドスタンダード（標準的指標）**とされている

## 2. 科学的な妥当性についての詳細

- **MBTIへの批判**:
  - 多くの研究で、MBTIは「再テスト信頼性」が低いことが指摘されている（5週間後に受けると約半数の人が異なる結果になるなど）
  - 人間の性格は「白か黒か」の二分法ではなく、正規分布（ベルカーブ）を描く連続的なものであるため、無理やり2つに分けること自体に無理があるとされる
- **16Personalitiesの立ち位置**:
  - 彼らはMBTIの用語を使用しているが、モデル自体はビッグファイブに基づいて再構築されているため、伝統的なMBTIよりは現代的なアプローチであると主張している
  - ただし、結果を「タイプ」として出力する点ではMBTIと同様の「二分法の限界」を内包している可能性はあるが、一般層への普及と親しみやすさでは成功している
