# エスカレイション！

```
[h1]マゾならとにかくやれ！エスカレイション！[/h1]

マゾ向け3Dビジュアルノベルは現時点でこれしかないはず！
文章・展開・ボイス・3Dモデル・モーション・ボリューム、いずれも文句ありません！
約5時間でクリアしましたが、モデル表情の変化やモーションのリアルさに感動しました！

[h2]あらすじ[/h2]

ご主人様になった彼女と倒錯していく物語。
最初は家で下僕として扱われる悦びを重視しているが、
彼女の行動がエスカレートしていき・・・

[h2]含まれる要素[/h2]

購入前の参考に。
前半→後半で徐々にハードになっていきます……！

[h3]前半[/h3]

[spoiler]
- 下僕・奴隷扱い（言葉責め、家事対応など）
- 包茎いじり（仮性）
- 乳首責め、手コキ
- 踏みつけ、足コキ、靴舐め、精液踏み
- 首輪、目隠し、縄、磔
- 蝋燭、鞭、ビンタ
- ノーハンド射精
- 唾液飲ませ、唾かけ
[/spoiler]

[h3]後半[/h3]

[spoiler]
- 指アナル、ペニバン
[/spoiler]

[h3]含まれない要素[/h3]

[spoiler]
- 騎乗位などの挿入
- SPH（「貧相」程度）
- 去勢、玉潰し
- 不可逆変化、死
[/spoiler]

[h2]総評[/h2]

3Dなので値段が少し張っていますが、私は定価でも高いとは思いません！
次回作があれば即買いレベルです！
```
